# FUTURE_EXTENSIONS

Planned upgrades and extension ideas

- Cloud deployment and orchestration (Kubernetes, Helm charts) for scalable agents and API services.
- Integrate external threat-intel APIs (TI feeds, MISP) to enrich Intel agent output.
- RAG (Retrieval-Augmented Generation) explainability for model decisions to produce human-friendly rationales.
- Role-Based Access Control (RBAC) for multi-user dashboards and safe approval workflows for response actions.
- Automated model retraining pipelines with data labeling feedback loops from the dashboard.
